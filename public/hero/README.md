# Hero (фон)

Для главной страницы используется **CSS-глитч** поверх статичного фона:

- `hero-bg.webp` — фото телевизора (`img/hero-bg.png` → `npm run optimize-images`)

Константы скролла и текста: `src/data/hero.js`.

Папки `hero-1`, `hero-2`, `transfer-*` — устаревшие спрайты canvas-hero, в сборке не используются (можно удалить из репозитория для уменьшения размера).
